Powerful Features

Everything you need to stay safe online

URL Lexical Analysis

Extracts 25+ features from URL structure including length, domain characteristics, special characters, and suspicious patterns.

  • URL/Domain/Path length
  • Count of dots, hyphens, underscores
  • IP address detection
  • HTTPS verification
  • Parameter and subdomain analysis

HTML Structural Features

Analyzes page HTML to detect malicious patterns like hidden forms, iframes, and external link redirects.

  • Form analysis
  • Hidden iframe detection
  • Internal/External link ratio
  • Cross-domain form actions
  • Script tag analysis

NLP Semantic Features

Uses advanced NLP techniques to analyze page text content for urgency keywords and semantic patterns.

  • TF-IDF vectors (50 dimensions)
  • Word2Vec/GloVe embeddings
  • BERT embeddings (768 dimensions)
  • Urgency keyword detection
  • Text sentiment analysis

Ensemble Stacking

Combines predictions from 4 different ML models using a meta-classifier for superior accuracy.

  • Random Forest (100 trees)
  • XGBoost
  • Bidirectional LSTM
  • Fine-tuned BERT
  • Logistic Regression meta-classifier
